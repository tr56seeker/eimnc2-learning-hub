# EIM NC II Learning Hub Roadmap

## Phase 1: MVP

Included in this starter:

- Login and signup
- Learner dashboard
- Teacher dashboard
- Lessons
- Exams
- Submissions
- Grades
- Supabase schema and RLS

## Phase 2: Testportal-style features

Recommended next build:

- Exam builder UI
- Question bank CRUD
- CSV/Excel question import
- Timer with auto-submit
- Randomized questions
- Randomized choices
- Item analysis
- Mastery level report
- Retake/remediation settings

## Phase 3: DepEd/EIM reports

- Export gradebook to Excel
- Export item analysis
- Export missing outputs report
- Export learner mastery report
- Section-level analytics
- Individual learner progress report

## Phase 4: Low-data learning mode

- Printable SLM view per lesson
- PWA install prompt
- Offline cached lessons
- Compressed image handling
- Google Drive/YouTube link-based submissions

## Phase 5: Admin controls

- Teacher-managed learner accounts
- Section management
- School year archiving
- Backup/export tools
- Audit logs
